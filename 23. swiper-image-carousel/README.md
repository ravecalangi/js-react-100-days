# Swiper Image Carousel

## Description
- A responsive horizontal image carousel that allows users to scroll through images using mouse wheel or navigation buttons.  
- Images are displayed in grayscale by default and highlight on hover.

## Features
- Horizontal scrolling with mouse wheel.  
- Smooth navigation using previous and next buttons.  
- Hover effect on images (grayscale to color with zoom).  
- Responsive grid layout for image groups.  

## Technologies Used
- HTML5  
- CSS3  
- JavaScript (ES6)  

## How to Run
1. Clone or download the project folder.  
2. Open `index.html` in any modern web browser.  

## Usage
1. Scroll horizontally through the gallery using your mouse wheel.  
2. Click the left (`back`) or right (`next`) buttons to navigate between image groups.  
3. Hover over any image to see a zoom and color effect.  

**Example:**
- Scroll down with mouse wheel → gallery scrolls horizontally.  
- Click `next` → gallery moves to the next set of images.  
- Hover on an image → it changes from grayscale to color and slightly enlarges.
