# Animal Search Autocomplete

## Description
A simple web application that provides real-time autocomplete suggestions for animals as the user types in the search input.

## Features
- Displays all animal suggestions immediately when the input is focused.  
- Filters and shows matching animal names as the user types.  
- Case-insensitive search.  
- Hides suggestions when the input is empty.  

## Technologies Used
- JavaScript (ES6)  
- HTML  
- CSS  

## How to Run
1. Download or clone the project.  
2. Make sure `index.html`, `style.css`, and `script.js` are in the same folder.  
3. Open `index.html` in your web browser.  

## Usage
1. Type any animal name (or part of it) in the search input.  
2. The app will display all matching animal names below the input.  
3. When the input is cleared, the suggestion list is hidden.  

**Example:**

- Input: `do`  
- Output:  
  - dog  

- Input: `p`  
- Output:  
  - penguin  
  - panda  
  - vampire bat
